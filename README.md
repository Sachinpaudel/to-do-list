# Todo App

A simple Todo App built using HTML, CSS, and JavaScript.

## Features

- Add todos
- Delete todos
- Edit todos
- Simple UI

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Installation

1. Clone the repository
2. Open `index.html` in your browser

## Usage

Type a task and click **Add**.


## Future Improvements

- Save todos in Local Storage
- Dark mode
- Due dates
- Search todos
- Responsive design
